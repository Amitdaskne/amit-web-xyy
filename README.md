# 🌐 Amit Web XYY

Chrome-like Android browser with all features.

## Features
- 🏠 Custom Homepage
- 📑 Multi-tab support
- ⭐ Bookmarks
- 🕐 History
- 🕶️ Incognito Mode
- 📥 Downloads manager
- 💻 Desktop/Mobile toggle
- 🔄 Pull to refresh
- 📤 Share URLs
- ⚙️ Settings

## Download APK
Go to **Actions** tab → Select workflow → Download from **Artifacts**
